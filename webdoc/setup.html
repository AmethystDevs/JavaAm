JavaAm Setup Guide

Install JavaAm, build the compiler, and configure the VS Code extension.

Home Project Language Cheatsheet Setup Contribute

Prerequisites

  • Java Development Kit (JDK) 11 or higher
  • Maven 3.6 or higher
  • Git (optional)
  • Visual Studio Code (optional)

Install Java

Verify Java installation using:

java -version
javac -version

Install Maven

Verify Maven installation using:

mvn -version

Build the JavaAm Compiler

cd ..
# Windows
build.bat

# macOS / Linux
chmod +x build.sh
./build.sh

# Manual build
cd compiler
mvn clean package

The compiler output is stored in:

compiler/target/javaam-compiler-1.0.0.jar

Verify Your Installation

java -jar compiler/target/javaam-compiler-1.0.0.jar help

Set Up VS Code Extension

Copy the `javaam-vscode` folder to your VS Code extensions directory and reload the editor.

  • Windows: %APPDATA%\Code\User\extensions
  • macOS: ~/.config/Code/User/extensions
  • Linux: ~/.config/Code/User/extensions

Compile an Example

java -jar compiler/target/javaam-compiler-1.0.0.jar compile examples/HelloWorld.jvm

Troubleshooting

  • If java is not found, install JDK and add it to PATH.
  • If mvn is not found, install Maven and add it to PATH.
  • Restart your terminal after updating environment variables.
  • Confirm you are using JDK 11+ and not just a JRE.

Use this guide to install JavaAm quickly and reliably.
